Порядок работы:

1. Сохранить к себе на компьютер файл signoff.exe. В эту же папку сохранить файлы Sign Off.pptx (это шаблон презентации) и WellInfo.xlsx(форма для заполнения слайдов)
2. Запустить файл signoff.exe -> Выбрать файл с рапортом (этот файл также должен находиться в этой же папке) -> Нажать "Закрыть"
3. Итогом работы файла является эксель файл с формами для заполнения слайдов (которые можно редактировать) и готовая презентация Powerpoint
